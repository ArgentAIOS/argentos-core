# aos-pinecone agent harness

Local CLI harness for the ArgentOS Pinecone connector.

## Environment

- `PINECONE_API_KEY`
- `PINECONE_INDEX_NAME`
- `PINECONE_INDEX_HOST`
- `PINECONE_NAMESPACE`
- `PINECONE_VECTOR_ID`
- `PINECONE_INDEX_DIMENSION`
- `PINECONE_METRIC`
- `PINECONE_CLOUD`
- `PINECONE_REGION`
- `PINECONE_API_BASE_URL`
- `PINECONE_API_VERSION`

python -m pytest tests/
