# aos-blacksmith agent harness

This harness provides the ArgentOS CLI wrapper for Blacksmith read surfaces.

Commands:

- `runner.list`
- `runner.status`
- `build.list`
- `build.get`
- `build.logs`
- `cache.list`
- `cache.stats`
- `usage.summary`
- `usage.billing`
- `capabilities`
- `config.show`
- `health`
- `doctor`

Credentials:

- `BLACKSMITH_API_KEY`
- Optional defaults: `BLACKSMITH_REPO`, `BLACKSMITH_RUN_ID`, `BLACKSMITH_WORKFLOW_NAME`, `BLACKSMITH_DATE_RANGE`, `BLACKSMITH_API_BASE_URL`
