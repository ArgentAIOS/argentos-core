# aos-woocommerce

WooCommerce connector scaffold for ArgentOS.

This connector targets the WooCommerce REST API v3 for:

- orders (list, get, create, update)
- products (list, get, create, update)
- customers (list, get, create)
- coupons (list, create)
- reports (sales, top sellers)

Write paths remain scaffolded until WooCommerce write workflows are reviewed and approved.

Docs used for this scaffold:

- https://woocommerce.github.io/woocommerce-rest-api-docs/
- https://woocommerce.com/document/woocommerce-rest-api/
