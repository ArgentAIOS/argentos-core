# Vercel Skill

Use `vercel_deploy` for Vercel operations.

Primary actions:

- `test_connection`
- `list_projects`
- `create_project`
- `add_domain`
- `list_deployments`

Required key: `VERCEL_API_TOKEN`.
Optional: `VERCEL_TEAM_ID`.
