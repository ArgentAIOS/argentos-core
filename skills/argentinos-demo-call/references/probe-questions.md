# Probe Questions By Pain Pattern

## Alert Blindness

- "How many alerts are ignored daily because they are noisy?"
- "What distinguishes an urgent alert from normal noise today?"
- "Who is accountable when an alert is missed?"

## Communication Overhead

- "Which updates are copy-pasted across tools?"
- "How many channels must be touched for one incident?"
- "Where does context get lost between teams?"

## Ticket/Queue Backlog

- "What ticket type grows faster than it closes?"
- "Where do tickets stall due to missing context?"
- "What would automatic triage need to decide correctly?"

## Executive Reporting Drag

- "How long does weekly reporting take today?"
- "Which sections are manual and error-prone?"
- "If reports were real-time, what decision cadence changes?"

## Compliance And Audit Risk

- "What evidence is hardest to reconstruct after the fact?"
- "Which process depends on memory instead of policy?"
- "What approval chain must be explicit in phase one?"

## Integration Gaps

- "What tools matter most but are not connected?"
- "Which API exists but is not used operationally?"
- "What one integration would remove the most manual work?"
