# Agent Proposal Language

## Proposal Frame

- "I am proposing this roster because each role maps to a pain point you named."
- "No placeholder agents. Every role has an accountable job."

## Agent Card Format

Use this format for each proposed agent:

1. Name
2. Domain responsibility
3. Trigger condition
4. Action scope
5. Human approval boundary

Example:

- "Dispatch Coordinator: owns inbound incident triage, triggered by new ticket events, classifies severity and routing, escalates human approval for customer-impacting actions."

## Confirmation Prompt

- "Which role should be phase one?"
- "Which role feels over-scoped or under-scoped?"
- "Is any critical responsibility missing?"

## Scope Guard

- "I will keep phase one to the smallest set that delivers measurable impact in week one."
