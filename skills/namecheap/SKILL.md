# Namecheap Skill

Use `namecheap_dns` for Namecheap API operations.

Primary actions:

- `test_connection`
- `check_domain`
- `get_hosts`
- `set_hosts`
- `raw`

Required keys:

- `NAMECHEAP_API_KEY`
- `NAMECHEAP_API_USER` (or `NAMECHEAP_USERNAME`)
- `NAMECHEAP_CLIENT_IP`
