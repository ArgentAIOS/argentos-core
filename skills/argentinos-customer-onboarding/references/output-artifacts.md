# Output Artifact Templates

Produce all four artifacts for each onboarding run.

## 1. Customer Strategy Document

Required sections:

- Executive summary
- Current-state pain map
- Target-state operating model
- Phase roadmap (1/2/3)
- Success metrics and ROI anchors
- Risks and mitigations

## 2. Technical Implementation Spec

Required sections:

- Architecture overview
- Agent roster with responsibilities
- Tooling/integration matrix
- Data and event flow
- Schedules/cron behaviors
- Guardrails and approval gates
- Rollout and rollback plan

## 3. Bootstrap Prompt

Required sections:

- System objective
- Agent identity and behavior
- Required workflows
- Constraints and never-do list
- Escalation and approval policy
- First-run checklist

## 4. Skills Gap Report

Required sections:

- Needed capabilities
- Existing capabilities
- Gap list with complexity
- Build order by phase
- Ownership and ETA assumptions
