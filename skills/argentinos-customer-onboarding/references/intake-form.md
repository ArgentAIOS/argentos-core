# Structured Intake Form (Pre-Session)

Capture before discovery session.

## Company Profile

- Company name
- Industry
- Employee count
- Locations/time zones
- Primary contact (name, role)
- Executive sponsor

## Operating Model

- Core departments
- Shift windows or service hours
- Priority customer segments
- Existing SOP documentation availability

## Current Stack

- Ticketing/helpdesk
- Email platforms
- Chat/collaboration tools
- Monitoring and alerting
- CRM/ERP/finance systems
- Knowledge/document stores

## Pain And Risk

- Top 5 pain points
- Most expensive recurring failure
- Compliance/legal constraints
- SLA/response commitments
- Current escalation path

## Integration Readiness

- Available APIs
- Authentication model
- Data export limits
- Security requirements
- Sandbox/test environment availability

## Desired Outcomes

- Week-one success metric
- 30-day success metric
- 90-day success metric
- One automation that must ship first
