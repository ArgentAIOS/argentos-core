# Twilio Skill

Use `twilio_comm` for Twilio messaging.

Primary actions:

- `test_connection`
- `list_numbers`
- `send_sms`
- `send_whatsapp`

Required keys:

- `TWILIO_ACCOUNT_SID`
- `TWILIO_AUTH_TOKEN`
  Optional:
- `TWILIO_FROM_NUMBER`
