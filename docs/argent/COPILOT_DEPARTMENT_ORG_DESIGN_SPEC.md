# Department + Org Design Co-Pilot Spec

## Problem

As workforce count grows, operators lack guidance on org structure quality.

## Goal

Help operators design clear department ownership and escalation structure.

## Scope

- department definitions
- role distribution
- overlap/gap detection
- escalation paths

## Co-Pilot Capabilities

- propose org structure from company goals
- identify overlap and ownership gaps
- recommend responsibility rebalance
- validate escalation chain coverage

## Required UI

- org map view
- overlap/gap diagnostics
- suggested rebalance actions

## Acceptance Criteria

1. Operator can identify ownership gaps and overlaps quickly.
2. Org changes are reviewed with impact preview.
3. Escalation paths are explicit and testable.

## Test Plan

- integration: org diagnostics + rebalance suggestions
