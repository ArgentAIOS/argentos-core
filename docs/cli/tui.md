# `argent tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/tui)

## Examples

```bash
argent tui
argent tui --url ws://127.0.0.1:18789 --token <token>
argent tui --session main --deliver
```
