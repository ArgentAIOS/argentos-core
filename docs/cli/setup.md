# `argent setup`

Initialize `~/.argentos/argent.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
argent setup
argent setup --workspace ~/.argentos/workspace
```

To run the wizard via setup:

```bash
argent setup --wizard
```
